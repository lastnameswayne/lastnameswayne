## Tiarnan Swayne Storgaard

Software engineer at Uber. Interested in systems programming, containers, and distributed systems.

Built [realm-flipper-plugin](https://github.com/realm/realm-flipper-plugin) at MongoDB, a Flipper plugin for debugging React Native Realm databases.

- [modal-runner](https://github.com/lastnameswayne/modal-runner) VS Code extension that adds run buttons above Modal functions. Rust backend (tree-sitter) + TypeScript frontend. [On the marketplace.](https://marketplace.visualstudio.com/items?itemName=swayne.modal-runner)
- [raft](https://github.com/lastnameswayne/raft) Raft consensus algorithm: leader election, log replication, persistence. Go.
- [tinyDB](https://github.com/lastnameswayne/tinyDB) Key value store implementing the Bitcask paper. Append only log structured storage with in memory index. Go.
- [mininearestneighbors](https://github.com/lastnameswayne/mininearestneighbors) HNSW approximate nearest neighbor search from scratch. Go.
- [git](https://github.com/lastnameswayne/git) Git reimplementation. Python.
